Short Source Quotes

The Marin model declares the architecture.

Architecture: QB-routed MoE with GatedNorm, XSA, sigmoid combine weights.
No load-balancing loss; router z-loss only. All layers are MoE.

QB is Quantile Balancing, sourced to an article-style update.

Open Athena: Quantile Balancing (QB), a load balancing method for Mixture of Experts.
Marin #3406: article-style MoE quantile load-balancing update from kexue.fm/archives/11619.

The router selects top-(K+1), then keeps top-K.

_topk_logits, selected_experts = jax.lax.top_k(..., K + 1)
selected_experts = selected_experts[:, :-1]

QB betas become next-step router bias.

new_bias = -qb_betas[moe_idx]
new_bias = new_bias - jnp.mean(new_bias)

Combine weights use sigmoid, not selected-expert softmax.

combine_weights = jax.nn.sigmoid(unbiased_topk).astype(x.dtype)

RMSNorm rescales by root-mean-square magnitude and learned weight.

variance = jnp.mean(jnp.square(x), axis=-1, keepdims=True)
normed = x * jax.lax.rsqrt(variance + self.eps)
return (normed * weight).astype(dtype)

GatedNorm multiplies by a learned sigmoid gate.

gate_hidden = jax.nn.silu(x @ w_down)
gate = jax.nn.sigmoid(gate_hidden @ w_up)
return x * gate.astype(x.dtype)

Every fourth layer gets the long attention window.

layer_mask = long_mask if i % 4 == 3 else short_mask

Attention normalizes Q/K, applies RoPE, and scales Q.

q = rms_norm(q)
k = rms_norm(k)
q, k = apply_rotary_embedding(...)
q = q * self.cfg.qk_mult

Exclusive self-attention removes the own-value direction.

dot = jnp.sum(attn_out * aligned_v, axis=-1, keepdims=True)
attn_out = attn_out - (dot / (v_norm_sq + 1e-6)) * aligned_v

Attention applies a learned headwise gate after XSA.

# Headwise gating: sigmoid(x @ attn_gate) produces one scalar per head.
gate = 2 * jax.nn.sigmoid(jnp.einsum("bsd,dn->bsn", x, self.attn_gate))[..., None]
attn_out = gate * attn_out

The shared expert runs beside routed experts when present.

if self.shared is not None:
    mlp_out = mlp_out + self.shared(..., activation=ActivationFunctionEnum.silu)
